# Neaboats — Broker náutico del NEA

Sitio web profesional para la compraventa de embarcaciones en el Nordeste Argentino.

🌐 **[Ver sitio en vivo](https://guileless-palmier-d886e3.netlify.app)**

---

## Descripción

Plataforma completa desarrollada para Neaboats, un broker náutico especializado en el NEA. El sitio permite a compradores explorar el catálogo de embarcaciones disponibles y a vendedores publicar sus lanchas de forma sencilla.

## Funcionalidades

- 🔍 Catálogo interactivo con filtros por precio, año y ubicación
- 📋 Fichas de detalle con modal para cada embarcación
- 📱 100% responsive — optimizado para móviles
- 💬 Contacto directo por WhatsApp integrado
- 📝 Formulario de publicación para vendedores
- ⚡ Animaciones y scroll reveal
